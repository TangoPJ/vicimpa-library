# Документация для @vicimpa/lib-vec2

Библиотека `@vicimpa/lib-vec2` предоставляет обширный набор утилит для работы с 2D-векторами. Эта библиотека предназначена для упрощения математических операций, трансформаций и утилитарных функций для манипуляции 2D-векторами.

## Типы

### Типы Vec2

- **Point (Точка)**
  ```typescript
  type Point = {
      x: number;
      y: number;
  };
  ```

- **Tuple (Кортеж)**
  ```typescript
  type Tuple = [x: number, y: number];
  ```

- **Size (Размер)**
  ```typescript
  type Size = {
      width: number;
      height: number;
  };
  ```

- **PageXY**
  ```typescript
  type PageXY = {
      pageX: number;
      pageY: number;
  };
  ```

- **OffsetXY**
  ```typescript
  type OffsetXY = {
      offsetX: number;
      offsetY: number;
  };
  ```

- **DeltaXY**
  ```typescript
  type DeltaXY = {
      deltaX: number;
      deltaY: number;
  };
  ```

- **OffsetSize**
  ```typescript
  type OffsetSize = {
      offsetWidth: number;
      offsetHeight: number;
  };
  ```

- **InnerSize**
  ```typescript
  type InnerSize = {
      innerWidth: number;
      innerHeight: number;
  };
  ```

- **Args (Аргументы)**
  ```typescript
  type Args = [xy: number | Point | Tuple] | Tuple;
  ```

- **ClampArgs (Аргументы для ограничения)**
  ```typescript
  type ClampArgs = [min: Args[0], max: Args[0]] | [minX: number, minY: number, maxX: number, maxY: number];
  ```

## Интерфейсы

### Базовые методы

- **set**: Устанавливает компоненты вектора.
- **equal**: Проверяет, равны ли два вектора.
- **clone**: Клонирует вектор.
- **toObject**: Преобразует вектор в объект `Point`.
- **toObjectSize**: Преобразует вектор в объект `Size`.
- **toString**: Преобразует вектор в строку.
- **[Symbol.iterator]**: Итерация по компонентам вектора.
- **[Symbol.toStringTag]**: Возвращает строковый тег вектора.

### Математические методы

- **plus**: Складывает векторы.
- **minus**: Вычитает векторы.
- **times**: Умножает векторы.
- **div**: Делит векторы.
- **rem**: Вычисляет остаток от деления векторов.
- **pow**: Возводит векторы в степень.
- **abs**: Вычисляет абсолютное значение вектора.
- **sign**: Вычисляет знак вектора.
- **round**: Округляет компоненты вектора.
- **ceil**: Применяет функцию округления вверх к компонентам вектора.
- **floor**: Применяет функцию округления вниз к компонентам вектора.

### Утилитарные методы

- **angle**: Вычисляет угол вектора.
- **length**: Вычисляет длину вектора.
- **min**: Находит минимальный компонент вектора.
- **max**: Находит максимальный компонент вектора.
- **distance**: Вычисляет расстояние между векторами.
- **inverse**: Инвертирует вектор.
- **normalize**: Нормализует вектор.

### Методы ограничения

- **clampMin**: Ограничивает вектор минимальным значением.
- **clampMax**: Ограничивает вектор максимальным значением.
- **clamp**: Ограничивает вектор между минимальным и максимальным значениями.

### Методы копирования математических операций

- **cplus**: Складывает векторы и возвращает новый вектор.
- **cminus**: Вычитает векторы и возвращает новый вектор.
- **ctimes**: Умножает векторы и возвращает новый вектор.
- **cdiv**: Делит векторы и возвращает новый вектор.
- **crem**: Вычисляет остаток от деления векторов и возвращает новый вектор.
- **cpow**: Возводит векторы в степень и возвращает новый вектор.
- **cabs**: Вычисляет абсолютное значение вектора и возвращает новый вектор.
- **csign**: Вычисляет знак вектора и возвращает новый вектор.
- **cround**: Округляет компоненты вектора и возвращает новый вектор.
- **cceil**: Применяет функцию округления вверх к компонентам вектора и возвращает новый вектор.
- **cfloor**: Применяет функцию округления вниз к компонентам вектора и возвращает новый вектор.

### Методы копирования утилитарных функций

- **cinverse**: Инвертирует вектор и возвращает новый вектор.
- **cnormalize**: Нормализует вектор и возвращает новый вектор.

### Методы копирования ограничения

- **cclampMin**: Ограничивает вектор минимальным значением и возвращает новый вектор.
- **cclampMax**: Ограничивает вектор максимальным значением и возвращает новый вектор.
- **cclamp**: Ограничивает вектор между минимальным и максимальным значениями и возвращает новый вектор.

## Классы

### Vec2

Класс `Vec2` предоставляет обширный набор методов для манипуляции 2D-векторами.

#### Конструктор

```typescript
constructor(...args: Vec2.Args | []);
```

#### Статические методы

- **fromAngle**: Создает вектор из угла.
- **fromRandom**: Создает вектор с случайными компонентами.
- **fromSrandom**: Создает вектор с случайными знаковыми компонентами.
- **fromSize**: Создает вектор из объекта `Size`.
- **fromDeltaXY**: Создает вектор из `DeltaXY`.
- **fromPageXY**: Создает вектор из `PageXY`.
- **fromOffsetXY**: Создает вектор из `OffsetXY`.
- **fromInnerSize**: Создает вектор из `InnerSize`.
- **fromOffsetSize**: Создает вектор из `OffsetSize`.
- **fromSvgLength**: Создает вектор из длин SVG.

### Vec2Map

Структура, похожая на карту, для хранения значений, связанных с векторами.

#### Методы

- **has**: Проверяет, есть ли вектор в карте.
- **get**: Извлекает значение, связанное с вектором.
- **set**: Связывает значение с вектором.
- **delete**: Удаляет вектор из карты.
- **clear**: Очищает карту.
- **forEach**: Итерация по карте.

### Vec2Set

Структура, похожая на множество, для хранения уникальных векторов.

#### Методы

- **has**: Проверяет, есть ли вектор в множестве.
- **add**: Добавляет вектор в множество.
- **delete**: Удаляет вектор из множества.
- **clear**: Очищает множество.
- **forEach**: Итерация по множеству.

## Примеры

### Основные операции с векторами

```typescript
import { vec2 } from '@vicimpa/lib-vec2';

const v1 = vec2(1, 2);
const v2 = vec2(3, 4);

v1.plus(v2); // v1 теперь (4, 6)
v1.minus(v2); // v1 теперь (1, 2)
v1.times(2); // v1 теперь (2, 4)
v1.div(2); // v1 теперь (1, 2)
```

### Использование Vec2Map

```typescript
import { Vec2Map, vec2 } from '@vicimpa/lib-vec2';

const map = new Vec2Map<number>();
map.set(vec2(1, 2), 100);
console.log(map.get(vec2(1, 2))); // Выводит: 100
```

### Использование Vec2Set

```typescript
import { Vec2Set, vec2 } from '@vicimpa/lib-vec2';

const set = new Vec2Set();
set.add(vec2(1, 2));
console.log(set.has(vec2(1, 2))); // Выводит: true
```
